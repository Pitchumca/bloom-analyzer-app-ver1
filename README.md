Bloom's-Level Balance Analyzer
Overview

Bloom's-Level Balance Analyzer is an AI-powered educational analytics platform developed to assist faculty members, academic administrators, and accreditation teams in evaluating the cognitive-level distribution of question papers based on Bloom's Taxonomy.

The system automatically extracts questions from DOC, DOCX, PDF, TXT, CSV, and Excel files, identifies Bloom's cognitive levels using a hybrid rule-based and machine learning approach, and generates comprehensive analytics on assessment quality.

Key Features
Automatic question extraction from multiple file formats
Bloom's Taxonomy classification (Remember, Apply, Analyze, Evaluate, Create)
Hybrid Rule-Based + Machine Learning classification engine
Question-wise cognitive level analysis
Bloom-level distribution visualization
Assessment balance evaluation
Higher-order thinking coverage analysis
Expert validation module with Accuracy and Cohen's Kappa metrics
Downloadable analytical reports
Interactive Streamlit web interface
